# 村户慧眼台账系统
### 基于航拍地图的村户台账可视化管理平台

> 燕山大学"智汇"黑客松 · 第 1 期参赛作品
> 出题方：燕山大学驻围场县银窝沟乡某村工作队

一个**纯前端、单机运行、双击即用、可整体拷走**的 Web 版村户信息系统。以村庄航拍俯视图为主界面，把住户位置与户主、家庭成员、联系方式、种养殖情况等台账信息一一对应，便于驻村干部**查得到、改得动、带得走**。

---

## 一、如何运行（无需安装、无需服务器、无需联网）

**方式一（推荐）**：双击根目录的 `启动.bat`，自动用默认浏览器打开。

**方式二**：直接双击 `index.html`。

> 推荐使用 Chrome / Edge 浏览器。所有数据、依赖库、照片均已内置在本文件夹，断网也能完整运行，数据不出本机。

---

## 二、数据规模（脱敏样例）

- 住户 **208 户**、家庭成员 **634 人**、覆盖 **6 个村民组**
- 自动关联 **人员照片**（按成员编号）与 **房屋照片**（按门牌"组+号"）
- 航拍底图 4 张（一组 / 二三五组 / 四六组 / 村部全景）+ 门牌标注图 3 张

## 三、核心功能（赛题要求 ①~④ 全部实现）

| # | 要求 | 实现 |
|---|------|------|
| ① | 数据导入与建档 | 解析脱敏 Excel，建立"户—成员—产业/养殖—位置"模型；自动关联脱敏照片与点位 |
| ② | 地图/航拍图展示 | 航拍俯视图上标注住户点位，点击点位查看该户台账详情，实现"地图点位—Excel 台账"对应 |
| ③ | 增删改查与筛选 | 户/成员的新增、编辑、删除；按姓名编号、电话、门牌、户类型、种养殖、重点人群多维搜索筛选 |
| ④ | 单机运行与可迁移 | 纯静态文件，双击即用；按表头关键字映射列，**其他村提供同类 Excel 即可一键生成系统** |

## 四、加分方向（选做 ①~⑤ 全部实现）

1. **数据统计与可视化**：汇报看板自动统计户数、人口、重点人群、种植/养殖规模，含分组柱状图、人群/产业环形图、重点人群名单。
2. **批量导入校验**：检查字段缺失、身份证/联系方式格式、门牌/经纬度异常、重复户/重复编号，分级展示问题清单；支持导入前 Excel 预检。
3. **便捷备份与导出**：一键 JSON 备份 / 恢复；导出 Excel 台账、PDF 汇报页、看板图片；提供空白模板下载。
4. **轻量权限与操作日志**：区分"查看 / 编辑"角色（编辑口令 `ysu2026`），关键修改全程记入操作日志。
5. **AI 辅助功能**：本地规则引擎（离线、不外传）实现照片标注自动识别关联、门牌/点位智能匹配纠错、入户走访摘要自动生成。

## 六、技术栈

原生 HTML + CSS + JavaScript（无前端框架，零构建）；
SheetJS（Excel 读写）、jsPDF + html2canvas（PDF/图片导出）；
数据持久化用浏览器 localStorage；图表用原生 Canvas 手绘。
数据生成脚本：Python + openpyxl。

## 七、从脱敏数据替换为真实数据 / 迁移到其他村

系统内置 **「数据替换」向导**（顶部页签），全程在本机浏览器完成，**无需命令行、无需联网**，村干部即可操作：

1. **① 选 Excel**：准备同结构 Excel（表头含"序号/户主/成员/关系/身份证号/性别/电话/门牌/种植/养殖/其他"，列顺序不限）。向导内可一键下载**空白模板**。
2. **② 选人照文件夹**：每张人照按**成员编号**命名（如 `T1_H_002_01.jpg`），系统自动对应到该成员。
3. **③ 选房照文件夹**：文件名含**组+门牌号**（如 `二组114号.jpg`），系统按门牌自动对应到该户。
4. **④ 选航拍底图（可选）**：文件名含"一组/二三五组/四六组/村部"即自动归位。
5. 查看**匹配预览**（人照/房照命中数），确认后点 **「确认替换为真实数据」**，一键完成。真实照片存入浏览器 IndexedDB，台账存 localStorage，刷新仍在。

> **跨电脑迁移 / 多村部署 / 长期备份**：在向导底部 **「导出完整数据包」** 可把全部台账与照片打包成单个文件，拷到另一台电脑后用 **「导入完整数据包」** 即可完整还原，**无需重新关联照片**。
>
> 校验逻辑已同时兼容脱敏与真实的身份证、手机号，真实数据不会被误判。

**附：批量重建静态包（可选，给有技术支持的场景）**：`tools/生成数据_generate.py` 路径已改为脚本相对、可移植——把数据放入 `第1期数据/` 后直接运行，即可重新生成可整体拷走的 `data/ledger-data.js` 与 `assets/`（也可用环境变量 `VHL_ROOT`/`VHL_XLSX` 指定数据位置）。

## 八、隐私与合规

全部使用组委会提供的**脱敏样例数据**；身份证、电话均为掩码形式。系统纯本地运行，**不联网、不上传、不外传**任何数据。
