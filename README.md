# 村户慧眼台账系统
### 基于航拍地图的村户台账可视化管理平台

> 燕山大学"智汇"黑客松 · 第 1 期参赛作品
> 出题方：燕山大学驻围场县银窝沟乡某村工作队

一个**纯前端、单机运行、双击即用、可整体拷走**的 Web 版村户信息系统。以村庄航拍俯视图为主界面，把住户位置与户主、家庭成员、联系方式、种养殖情况等台账信息一一对应，便于驻村干部**查得到、改得动、带得走**。

---

## 一、快速开始（无需安装、无需联网、数据不出本机）

| 方式 | 操作 | 适用场景 | 能否写回文件 |
|---|---|---|---|
| **方式一（推荐）** | 双击根目录 `启动.bat` | 日常使用、演示、录入真实数据 | ✅ 改动实时写回 `data/ledger-data.js` 与 `assets/` |
| **方式二（应急）** | 直接双击 `index.html` | 临时查看、无 PowerShell 环境 | ⚠️ 改动仅存浏览器，刷新仍在但不落盘 |

**两种方式的区别**：

- `启动.bat` 会用 Windows 自带的 PowerShell 拉起一个**本地小助手**（`server.ps1`，监听 `127.0.0.1` 回环地址，仅本机可访问），再自动打开浏览器。此时录入、导入、改照片等会**实时写回本文件夹的真实文件**，关掉浏览器后小助手窗口约 25 秒自动退出。**不需要管理员权限、不需要联网、不占用对外端口。**
- 直接双击 `index.html` 走 `file://` 协议，浏览器安全策略不允许写本地文件，系统自动降级：台账存浏览器 `localStorage`、照片存 `IndexedDB`，**功能完整、刷新不丢**，只是改动不会同步回 `ledger-data.js`。

> 推荐 Chrome / Edge 浏览器。所有数据、依赖库、照片均已内置在本文件夹，**断网也能完整运行，数据不出本机。**

## 七、技术栈

- 原生 HTML + CSS + JavaScript（无前端框架，零构建）
- SheetJS（Excel 读写）、jsPDF + html2canvas（PDF / 图片导出）
- **三层数据存储**：台账存浏览器 `localStorage`；真实照片存 `IndexedDB`（台账里仅存 `idb:` 轻量引用，避免撑爆 localStorage）；经 `启动.bat` 打开时再由本地小助手把整库写回 `data/ledger-data.js` 与 `assets/` 真实文件
- 图表用原生 Canvas 手绘；数据生成脚本：Python + openpyxl

## 八、从脱敏数据替换为真实数据 / 迁移到其他村

系统内置 **「数据替换」向导**（顶部页签），全程在本机浏览器完成，**无需命令行、无需联网**，村干部即可操作：

1. **① 选 Excel**：准备同结构 Excel（表头含"序号/户主/成员/关系/身份证号/性别/电话/门牌/种植/养殖/其他"，列顺序不限）。向导内可一键下载**空白模板**。
2. **② 选人照文件夹**：每张人照按**成员编号**命名（如 `T1_H_002_01.jpg`），系统自动对应到该成员。
3. **③ 选房照文件夹**：文件名含**组+门牌号**（如 `二组114号.jpg`），系统按门牌自动对应到该户。
4. **④ 选航拍底图（可选）**：文件名含"一组/二三五组/四六组/村部"即自动归位。
5. 查看**匹配预览**（人照/房照命中数），确认后点 **「确认替换为真实数据」**，一键完成。

> **跨电脑迁移 / 多村部署 / 长期备份**：在向导底部 **「导出完整数据包」** 可把全部台账与照片打包成单个文件，拷到另一台电脑后用 **「导入完整数据包」** 即可完整还原，**无需重新关联照片**。
>
> 校验逻辑已同时兼容脱敏与真实的身份证、手机号，真实数据不会被误判。

**附：批量重建静态包（可选，给有技术支持的场景）**：`tools/生成数据_generate.py` 路径已改为脚本相对、可移植——把数据放入 `第1期数据/` 后直接运行，即可重新生成可整体拷走的 `data/ledger-data.js` 与 `assets/`（也可用环境变量 `VHL_ROOT` / `VHL_XLSX` 指定数据位置）。

## 九、角色与权限

- 顶部右上角徽标显示当前角色，默认 **「访客（查看）」**，仅浏览、搜索、导出，不能改数据。
- 点击徽标 → 输入编辑口令 **`ysu2026`** → 进入 **「编辑」** 角色，方可新增/编辑/删除户与成员、拖动点位、替换数据。
- 编辑角色的关键修改会自动记入 **「操作日志」** 页签。

## 十、常见问题（FAQ）

- **双击 `启动.bat` 弹出黑窗口一闪而过？** 属正常现象，小助手以最小化方式在后台运行，浏览器会自动打开；用完关掉浏览器，窗口约 25 秒后自动退出。
- **改了数据但 `ledger-data.js` 没变化？** 说明是用 `file://`（直接双击 index.html）打开的，写回功能未启用。改动安全存于浏览器，换用 `启动.bat` 打开即可实时落盘。
- **照片不显示？** 多为浏览器禁用了 `IndexedDB` 或处于隐身模式；换 Chrome / Edge 正常窗口打开即可。
- **想恢复到最初的演示数据？** 「备份导出」页签点 **「恢复初始数据」**，会从 `ledger-data.seed.js` 还原。
- **端口被占用启动失败？** 小助手会在 `8732–8780` 间自动挑选空闲端口，一般无需干预。

## 十一、隐私与合规

全部使用组委会提供的**脱敏样例数据**；身份证、电话均为掩码形式。系统纯本地运行，本地小助手只监听回环地址 `127.0.0.1`，**不联网、不上传、不外传**任何数据，也不对外暴露端口。
